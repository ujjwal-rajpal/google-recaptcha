== Description ==

A google recaptcha for user login admin page.One can use it to authenticate a bot.It can be used only at admin login page.

== Installation ==
 
This section describes how to install the plugin and get it working.
1. Create a folder googlerecaptcha in  `/wp-content/plugins/` directory
2. Upload  to the files in  it.
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where did i get the site key and site secret?

Ans: steps to followed:
	-> Go to site https://www.google.com/recaptcha/admin
	-> Register for captcha and you will get your site key and secret
	-> Enter the key and secret in setting of plugin
	
	* Be carefull while entering key and secret
